jQuery-modal

经过IE7&IE6+，chrome，firefox测试兼容(不兼容ie6)

HTML

  1. 无需更改

CSS

  1. 无需更改

JS

  1. <script type="text/javascript" src="lib/jquery-1.9.1.js"></script>
  2. <script type="text/javascript" src="jquery.slider.js"></script>

调用

  1. $.modal();
选项
  1. $.modal({
  2.     'dialogWidth' : 500, //弹出对话框的宽度，默认为300
  3.     'dialogContent' : @html, //对话框的内容：传入你要放在对话框中的html
  4.     'hideDelay' : 2000,//对话框自动隐匿时间。默认值是'none'，即不会自动隐匿。你可以填入数值（ms单位）
  5.     'showCloseButton' : 0 //对话框是否带“关闭”按钮:1为带有，0为没有。默认带有。
  6. });
